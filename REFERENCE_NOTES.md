# 参考文档核心要点提取

## 一、《蓉才通×HigherMatch 中小微招聘信任提效合作方案》

### 1.1 产品定位
- "从出生就不是一个招聘工具，而是一个可以被信任的决策伙伴"
- 把CA强实名转化为中小微招聘的"信任货币"
- 独有资产：U盾/CA强实名（企业真实性权威背书）+ 60万已注册中小微企业流量池
- 核心机制：信任传导 CA→认证企业(挂认证标识)→真实岗位→候选人逐级背书

### 1.2 三重痛点
| 主体 | 痛点 |
|------|------|
| 中小微企业(招聘方) | 没有雇主品牌,候选人默认不信;招人难,爽约多,留人难 |
| 候选人(求职方) | 不敢投小公司,怕虚假岗位,信息不对称;决策孤立无援 |
| 平台(贵方) | 海量企业仅用于认证,未转化为高黏、有粘性的招聘服务与增值收入 |

### 1.3 信任传导架构——五步链路
| 步骤 | 角色 | 发生了什么 |
|------|------|-----------|
| ①凭证 | U盾/数字证书 | 贵平台CA颁发,企业以硬件凭证强实名登录 |
| ②校验写入 | CA认证 | 验证通过,实名身份写入企业信用底座 |
| ③背书 | 认证企业 | 企业获「CA认证真实企业」认证标识 |
| ④挂载真实内容 | 真实岗位UGC | 认证企业发布岗位真实答疑/招聘真实心声 |
| ⑤信任落地 | 候选人 | 看到「认证真企业说真话」,信任成本断崖下降 |

**一句话**: CA信任,经决策社区,变成了中小微招聘的「信任货币」。技术上,CA校验收敛于单一接口,可由演示无缝替换为贵平台真实政务网关,不影响业务流程。

### 1.4 双端价值
**企业侧(招聘方)**:
- CA认证标识：真实企业身份,候选人一眼可信
- 岗位真实答疑：公开真实工作内容/薪资结构,降低沟通成本,提升到面率
- 同行互助圈：中小微HR之间交流招聘与留人经验,提升粘性
- 招聘提效看板：以数据看见「省了多少,招得多快」,支撑续费与增值

**候选人侧(求职方)**:
- CA真实企业可辨：知道是认证真企业,真岗位,敢投敢面
- 决策陪伴：选offer/转行/纠结时,有同处境的人和真实数据可参考,不再孤立
- 信息透明：岗位真实信息公开,减少入职预期落差

### 1.5 跨端互证(单边平台做不到)
企业真(CA背书) + 候选人真(履约与社区信用),双向消除信息差。这是BOSS/猎聘等单边、流量驱动平台无法提供的。

### 1.6 合规与数据安全
| 维度 | 保障措施 |
|------|----------|
| 实名底座 | 基于U盾/CA强实名,从源头杜绝虚假企业与刷量 |
| 数据隔离 | 企业与候选人数据严格分离;企业专属内容仅认证企业可见,模选人无权访问 |
| 内容合规 | 内容审核前置,脱敏处理;实名理性社区,不设匿名情绪宣泄通道 |
| 权限控制 | 岗位/评论按角色过滤,越权访问一律拒绝,操作可审计,可追溯 |
| 政务级安全 | 按等保护要求建设;CA校验可对接贵平台真实政务网关 |
| 信创可适配 | 支持国产化部署路径(国产CPU/OS/数据库),满足自主可控要求 |

**说明**: 匿名情绪类内容在本平台默认关闭,情绪陪伴功能仅在后续移动端独立评估,确保政务平台内容可控、舆情可控。

### 1.7 分期落地
| 阶段 | 周期(参考) | 内容 | 双方分工 |
|------|-----------|------|----------|
| P0试点 | 约4-6周 | 小范围认证企业+认证标识+岗位真实答疑跑通 | 贵方CA对接/种子企业;我方:产品/部署 |
| P1提效 | 约2-3月 | 决策社区+信用体系+招聘提效看板上线 | 贵方:流量/合规;我方:AI/运营支持 |
| P2规模化 | 约3-6月 | 规模推广+跨端互证+数据增值服务 | 双方联合运营/增值分成 |

### 1.8 预期效果指标
| 指标 | 基线 | 目标区间 | 度量方式 |
|------|------|----------|----------|
| 岗位到面率 | 行业基线【待验证】 | +15%~30%【待验证】 | 试点企业到面/邀约比 |

| 平台增值收入 | 0(认证方式) | 新增招聘/数据增值【待验证】 | 增值服务流水 |

### 1.9 P0试点工作说明(SOW)
- **试点目标**: 验证CA信任传导链路跑通,并产出首批招聘提效数据用于校准
- **范围(In)**: 30-50家认证企业准入、CA认证标识、岗位真实答疑、B端基础决策社区、招聘提效看板
- **范围(Out)**: 匿名情绪社区、移动端、规模化推广、复杂变现、真实资金结算
- **周期**: 约4-6周
- **里程碑**:
  - W1: CA接口对接+种子企业准入 → 认证企业可经U盾/CA登录
  - W2-W3: 认证标识+岗位真实答疑上线 → 企业蓝标身份+首批真实岗位内容
  - W4: B端决策社区+提效看板上线 → 同行圈+数据看板可用
  - W5-W6: 数据采集与复盘 → 首批到面/转化数据+复盘报告
- **验收标准**:
  - 链路跑通: U盾登录→CA认证→企业发岗位答疑→候选人可见,全流程无阻断
  - 规模达标: ≥30家认证企业激活并产出真实内容【目标值待商定】
  - 数据产出: 产出首批到面率/转化对比数据,可用于P1目标校准
  - 合规零事故: 数据隔离高、内容审核、权限控制在试点期间零违规

### 1.10 已就绪的基础
- AI招聘引擎(简历筛选/AI面试/智能寻访/候选人助手/招聘工作流)双端可运行原型
- 决策社区与CA信任传导能力已完成原型与可演示版本,可现场演示
- CA校验接口已预留,可对接贵平台真实政务网关

- 候选人信任成本↓
